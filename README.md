# RK Group Passenger Service Excellence Training

Standalone HTML5 training presentation for RK Group's Passenger Service Excellence & IRCTC Compliance Training Program.

## Open

Open `index.html` in a browser, or run a small local server from this folder:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Controls

- `Next` / `Back` buttons move between sections.
- Number buttons on the left jump to any section.
- Use the `Subtitle` selector to show English only, or English with Hindi / Bengali subtitle support.
- Keyboard: `ArrowRight`, `ArrowLeft`, `PageDown`, `PageUp`, `Home`, `End`.
- Press `F` or the fullscreen icon for presentation mode.
- Click cards, steps, and equipment items for explanations.
- Complete the quiz to see the score and certificate message.

## Included Assets

The deck uses the provided RK Group / IRCTC training posters and real uniform / QR ID card images from the project brief. All required files are inside the `assets` folder so the presentation works offline.
